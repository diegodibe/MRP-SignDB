# MRP-SignDB
Script to download, segment and align sign language videos
it is possible to run the modules separately through console input.

If the scraper module does not work, please check whether the URL is correct.

in the folder Audio_to_text it is necessary to add the LIUM jar 'LIUM_SpkDiarization-8.4.1', please dowload it from https://projets-lium.univ-lemans.fr/spkdiarization/download/
